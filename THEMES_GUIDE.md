## Technical Details

### Theme Structure

Each theme includes:
```typescript
interface Theme {
  id: ThemeId;
  name: string;
  description: string;
  preview: string;
  colors: ThemeColors;      // 11 color variables
  typography: ThemeTypography;
  layout: ThemeLayout;
}
```

### CSS Variables

Themes generate CSS variables:
```css
--theme-primary
--theme-primary-hover
--theme-accent
--theme-background
--theme-background-alt
--theme-text
--theme-text-light
--theme-text-muted
--theme-border
--theme-card
--theme-glass
--theme-border-radius
--theme-font-display
--theme-font-body
```
